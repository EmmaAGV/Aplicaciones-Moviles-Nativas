# RoutinePro 🏋️‍♂️📊

RoutinePro es una aplicación de Android nativa diseñada para ayudar a los usuarios a gestionar y visualizar sus rutinas de ejercicio diarias de manera sencilla, elegante y eficiente.

## 🚀 Características

*   **Encuesta Personalizada**: Registro detallado de edad, peso, estatura, género y objetivos (adelgazar, ganar músculo, reducir estrés).
*   **Dashboard Inteligente**: Panel de bienvenida con gráfica de proyección de resultados y seguimiento de hidratación (agua).
*   **Plan de Bienestar Integral**: Mezcla de rutinas de ejercicio adaptadas al nivel y guías nutricionales completas.
*   **Workout Player**: Cronómetro de ejercicio y descanso con señales sonoras y guías visuales.
*   **Exploración de Biblioteca**: Catálogo de más de 50 ejercicios y guías de alimentación con categorías (Fuerza, Cardio, Nutrición).
*   **Estadísticas de Avance**: Gráficas de volumen semanal y registro de medidas corporales.
*   **Perfil Personalizado**: Gestión de avatar con carga de fotos desde la galería y metas actuales.
*   **Persistencia Local**: Uso de Room Database para guardar todo tu historial de forma segura.

## 🛠️ Stack Tecnológico

*   **Lenguaje**: Kotlin
*   **Arquitectura**: Jetpack Navigation, Fragments, ViewModel, LiveData.
*   **Interfaz Gráfica**: XML con Material Design 3 (Material 3).
*   **Componentes**: 
    *   ConstraintLayout (Diseños flexibles).
    *   ViewBinding (Vinculación de vistas segura).
    *   Room (Base de datos local).
    *   MPAndroidChart (Gráficas interactivas).
*   **Herramienta de Construcción**: Gradle (Kotlin DSL).

## 📥 Instalación

1.  Clonar el repositorio.
2.  Abrir en Android Studio (Koala o superior recomendado).
3.  Esperar a que Gradle sincronice las dependencias.
4.  **Nota para Xiaomi**: Desactivar "Optimización MIUI" en opciones de desarrollador para evitar errores de instalación.
5.  Ejecutar en un emulador o dispositivo físico (Android 7.0+).
